# Sakthi | Portfolio & Wave

Personal portfolio site with an embedded AI assistant (Wave), built with vanilla HTML/CSS/JS.

## Features
- Portfolio mode — pre-written answers about experience, skills, and contact
- Wave mode — live AI assistant powered by Groq + RAG pipeline
- Web search — prefix `\web` to search the internet in real time
- Document upload — ask questions about any PDF/DOCX you upload
- Light/dark theme toggle

## Tech Stack
- **Frontend** — Vanilla HTML, CSS, JS
- **Backend** — FastAPI, LangChain, Groq, Qdrant, HuggingFace Spaces
- **AI** — RAG pipeline, semantic search, streaming responses, Tavily web search
