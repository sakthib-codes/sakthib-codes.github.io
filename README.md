# Sakthi | Portfolio & Wave

Personal portfolio site with an embedded AI assistant (Wave), built with vanilla HTML/CSS/JS.

## Features
- Portfolio mode — pre-written answers about experience, skills, and contact
- Wave mode — live AI assistant powered by Groq + RAG pipeline
- Web search — prefix `\web` or ask naturally, Wave decides automatically
- Document upload — ask questions about any PDF/DOCX you upload
- LangGraph agent — auto-routes queries to web search, RAG, or general LLM
- MCP tool server — decoupled search_documents and search_web tools
- Light/dark theme toggle

## Tech Stack
- **Frontend** — Vanilla HTML, CSS, JS
- **Backend** — FastAPI, LangChain, LangGraph, Groq, Qdrant, HuggingFace Spaces
- **AI** — Agentic RAG pipeline, semantic search, streaming responses, Tavily web search, MCP architecture
